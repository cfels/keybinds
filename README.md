## Keybinds
a simple cpp program that shows keybinds for my dotfiles

(why did i use c++ and not bash? idk honestly i just felt like it was better to implement with c++)

### Compile
```
mkdir build && cd build
clear && cmake .. && cmake --build .
```

### dotfiles link
link to dotfiles: [cfels/nixos-dots](https://github.com/cfels/nixos-dots)
